# Geometry-Simulator
合同と相似シミュレーター (Geometry Simulator for Congruence and Similarity)Pythonのtkinterを使用した、図形の合同（Congruence）と相似（Similarity）の概念を視覚的に学べるインタラクティブな教育用シミュレーターです。

スライダーを動かすことで、図形がどのように変化（または維持）されるかをリアルタイムに確認できます。
💡 機能・学習のポイント画面上には3つの三角形が表示され、コントロールパネルから動的に操作できます。
図形変形タイプ特徴・学習ポイント基準の三角形 A (赤)固定すべての比較の基準となる図形です。
合同な三角形 B (青)合同変形平行移動や回転を行っても、形と大きさ（3辺の長さ・3つの角の大きさ）が完全に維持されることを確認できます。
相似な三角形 C (緑)相似変形基準の三角形と中心を重ねたまま拡大・縮小します。
大きさが変わっても図形の形状（各角の大きさの比率）が常に一定に保たれることが一目で理解できます。
🛠️ 動作環境Python 3.x標準ライブラリ（tkinter, math）のみを使用しているため、外部パッケージ（pip等）の追加インストールは不要です。
🚀 使い方1. プログラムの実行ターミナルまたはコマンドプロンプトを開き、スクリプトが保存されているディレクトリで以下のコマンドを実行します。
Bashpython GeometrySimulator.py
2. 操作方法平行移動 X / Y スライダー: 青い三角形（合同）を縦横に動かします。
回転角度 スライダー: 青い三角形（合同）をその場を中心に 0° 〜 360° 回転させます。
拡大・縮小倍率 スライダー: 緑の三角形（相似）のサイズを 0.2倍 〜 2.5倍 まで変化させます。
